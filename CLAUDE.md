# 百問計算 記録・グラフ化 Webアプリ

## プロジェクト概要
小学校の学級で毎日実施している「百問計算」の記録・グラフ化を行うWebアプリ。
Google Apps Script (GAS) + Google スプレッドシートをバックエンドとして使用。

## 技術スタック
- **バックエンド**: Google Apps Script (GAS)
- **フロントエンド**: HTML / CSS / JavaScript（GAS HTMLService）
- **グラフライブラリ**: Chart.js v4.x (CDN)
- **Excelライブラリ**: SheetJS (CDN)
- **データ**: Google スプレッドシート

## ファイル構成
- `コード.gs` — GASバックエンド（認証・データ読み書き・集計）
- `index.html` — フロントエンド（ログイン・児童UI・教師UI・グラフ）

## デプロイ手順
1. GASスクリプトエディタに `コード.gs` と `index.html` をコピー
2. スプレッドシートに「ユーザー」「記録」シートを手動作成
3. デプロイ → 「ウェブアプリ」として公開
4. 「アクセスできるユーザー」を適切に設定（全員 or 組織内）

## スプレッドシートシート構成
### ユーザーシート (ヘッダー行なし)
- A: 出席番号, B: 氏名, C: ログインID, D: パスワード, E: 権限(教師/児童)

### 記録シート (ヘッダー行なし)
- A: タイムスタンプ, B: 出席番号, C: 種目, D: プリント番号, E: 正答数, F: タイム(秒)

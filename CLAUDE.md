# 百問計算 記録・グラフ化 Webアプリ

## プロジェクト概要
小学校の学級で毎日実施している「百問計算」の記録・グラフ化を行うWebアプリ。
Google Apps Script (GAS) + Google スプレッドシートをバックエンドとして使用。

## 技術スタック
- **バックエンド**: Google Apps Script (GAS)
- **フロントエンド**: HTML / CSS / JavaScript（GAS HTMLService）
- **グラフライブラリ**: Chart.js v4.x (CDN)
- **Excelライブラリ**: SheetJS (CDN)
- **データ**: Google スプレッドシート

## 開発履歴

- **Phase 1**: GASバックエンドの基本ルーチン（シート初期化、CRUD操作）実装
- **Phase 2**: フロントエンドUI（ログイン画面、児童用入力画面、教師用名簿管理画面）実装
- **Phase 3**: パスワード管理、Chart.js連携でのグラフ描画、レコード一覧表示、Excel読み込み機能（SheetJS）追加
- **Phase 4**: 【大型改修】記録の表示期間フィルター・ログイン方式の変更（出席番号＋名前表示）・ログアウト時のクリア処理、および**「ログインID」の全面廃止・出席番号での一元管理化**を実施
- **Phase 5**: **自動更新機能（ポーリング）と手動更新ボタン**の追加。教師用・児童用両方の画面で、他者の更新情報がリアルタイムに近い形で同期されるようにUIと通信ロジックを強化。

## ファイル構成
- `コード.gs` — GASバックエンド（認証・データ読み書き・集計）
- `index.html` — フロントエンド（ログイン・児童UI・教師UI・グラフ）

## デプロイ手順
1. GASスクリプトエディタに `コード.gs` と `index.html` をコピー
2. スプレッドシートに「ユーザー」「記録」シートを手動作成
3. デプロイ → 「ウェブアプリ」として公開
4. 「アクセスできるユーザー」を適切に設定（全員 or 組織内）

## データベース設計（スプレッドシート）

### 1. ユーザーシート (SHEET_USERS)
| 列名 | 番号 (1-origin) | 説明 |
|---|---|---|
| 出席番号 | 1 (A) | 生徒の出席番号（ログインに使用、教師は `0` 等） |
| 氏名 | 2 (B) | 生徒または教師の正式名称 |
| パスワード | 3 (C) | ログイン用のパスワード（初回ログイン時に設定） |
| 権限 | 4 (D) | `教師` または `児童` |

### 2. 記録シート (SHEET_RECORDS)
- A: タイムスタンプ, B: 出席番号, C: 種目, D: プリント番号, E: 正答数, F: タイム(秒)
